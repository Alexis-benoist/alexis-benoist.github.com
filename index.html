Open Source

  • Wand is a ctypes-based simple ImageMagick API binding for Python.
  • Earth Reader is a feed reader which runs on various devices and is synchronized without any particular centralized proprietary services.
  • Geofront and geofront-cli are a simple SSH key management service and its client utility.
  • libsass-python is a SASS/SCSS package for Python.
  • sphinxcontrib-httpdomain is a Sphinx domain for describing HTTP APIs.
  • SQLAlchemy-ImageAttach is a SQLAlchemy extension for attaching images to entity objects.
  • Pixiv for Muzei is a Muzei extension sourcing artworks of Pixiv.
  • sphinxcontrib-autoprogram is a Sphinx extension for documenting CLI programs in automated way.
  • Et cetera:
    • Sider is a Python persistent object library using Redis.
    • Okydoky is an automated docs builder using Sphinx/GitHub/Distribute for use with private projects.
    • TypeQuery is a Python library to define generic methods to existing types
    • niconico-translator is a comment translator of Nico Nico Douga (ニコニコ動画).
    • NaverComicFeed는 네이버 만화를 위한 비공식 RSS 서비스입니다.
    • wsgi-proxy is a simple WSGI app that does HTTP 1.0 proxy.
    • irclog is a web IRC log viewer.
    • Lisphp is a Lisp dialect written in PHP.
    • encodingcontext is a small module that implements a bad idea, making default encoding explicitly scoped.
    • futureutils is a Python module that introduces future and promises into iterators and generators.
    • SQLAlchemy-Future is a SQLAlchemy extension that introduces future and promises into query.
